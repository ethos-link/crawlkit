# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-23

### Added

- Initial public release of `crawlkit`
- Sitemap-driven audits for metadata, structured data, uniqueness, and internal links
- Structured issues and grouped console reporting
- Optional browser rendering with Ferrum
- Rails rake tasks for sitemap audits and direct structured-data URL checks
- Zeitwerk-based loading and configurable schema registry support
